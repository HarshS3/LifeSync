# LifeSync AI Service

This microservice provides advanced AI features (conversational memory, RAG, suggestions) for the LifeSync app. Built with Python, ready for deployment (e.g., Render).

## Features
- Conversational memory (context-aware chat)
- Retrieval-Augmented Generation (RAG) with your docs
- Habit suggestions and AI coaching

## How to use
- Deploy this folder as a separate service (e.g., on Render)
- Expose endpoints for chat, suggestions, and RAG
- Call these endpoints from your Node.js backend

## Tech stack
- FastAPI (web framework)
- LangChain, LlamaIndex, or similar for AI
- OpenAI or other LLMs
- Vector DB (optional, for RAG)
